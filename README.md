# ☕ Java – String Manipulation Practice (Módulo 12)

Este repositório contém exercícios práticos focados na **manipulação de Strings em Java**, desenvolvidos durante o **Módulo 12 da formação Java da Alura**.

O objetivo deste módulo é praticar operações comuns com textos, explorando métodos da classe `String` e aplicando lógica para manipulação de dados textuais.

---

## 📚 Conceitos praticados

Durante os exercícios deste módulo foram explorados conceitos como:

* 🔤 Manipulação de **Strings**
* 🔍 Busca de palavras em textos
* ✂️ Uso de métodos como:

  * `contains()`
  * `substring()`
  * `toLowerCase()` / `toUpperCase()`
  * `trim()`
  * `length()`
* 🔁 Comparação de Strings
* 📄 Processamento de entrada de texto

Essas operações são fundamentais para lidar com **dados textuais em aplicações Java**, como validações, buscas e formatação de dados.


📌 Este projeto representa uma etapa de prática focada na **manipulação de Strings em Java**, um conceito essencial para trabalhar com entrada de dados e processamento de texto em aplicações.
